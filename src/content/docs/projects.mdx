## Web Development

### RoktoBondhu — Emergency Blood Donation Platform
**Tech:** Next.js 16, React 19, Supabase, TypeScript, Tailwind CSS

Emergency blood donation platform connecting donors and recipients in real-time with location-based matching.

🔗 [GitHub](https://github.com/fahimkarim01/roktobondhu) · [Live Demo](https://roktobondhu-bd.vercel.app/)

**Key Features:**
- Real-time donor-recipient matching
- Cookie-based authentication (@supabase/ssr)
- Row-Level Security (RLS) policies
- Mobile-responsive design
